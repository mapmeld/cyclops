Cyclops Lang.org xHELPx



Cyclops or 'xHELPx' is a programming language written in Linear A, a forgotten/undeciphered script used 3500-4500 years ago on the Greek island of Minos.

If you want to get into serious retro-programming, travel through time while teaching computer science, or make your code incomprehensible, Cyclops is the right language for your project.

Types

Commands are individual symbols in Linear A. Multiple characters can be the name of a variable or function. Uses Unicode codepoints for Linear A and Aegean font by George Douros.

Numbers are written in Aegean numerals.

Any non-Minoan characters are interpreted as strings.

Commands

  • xHELPx = help
  • xPRINTx = print
  • xINPUTx = receive user input (command line or input())
  • xRANDx = random number (1-100)
  • xLOOPx = loop until flag is raised
  • xBREAKx = raise flag (exit loop)

    • Math

    Math is evaluated left-to-right, without order of operations. So 3 + 2 * 4 == 20 not 11. It might be easier to do one math operation at a time.
